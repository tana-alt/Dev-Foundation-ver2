# Harness Tool: issue-create

## When
Escalate a problem that cannot be resolved algorithmically from repo evidence.

## How
Use only for real escalation. Routine bugs or TODOs should be fixed directly.

## What
Creates or dry-runs a GitHub issue according to policy.
