# Residual Risk Carryover

Status: retired from default workflow.

Do not use residual-risk records to make unfinished work look complete. For
ordinary work, state remaining risk directly in the final answer or plan log.

Use this shape instead:

```text
Not done:
Why:
Impact:
Next action:
```

Only inspect archived carryover records when the user explicitly asks for that
archive or migration work.
