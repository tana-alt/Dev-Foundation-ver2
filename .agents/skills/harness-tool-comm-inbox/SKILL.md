# Harness Tool: comm-inbox

## When
Read local task-scoped messages for a specific agent.

## How
Confirm `FOUNDATION_AGENT_ID` or pass the exact agent id. Follow `harness-acp-communication`.

## What
Returns local coordination messages without changing authority state.
