AGENTIC-FLOW obsidian_codex_workflow_starter provides an Obsidian × Codex workflow starter. Key structure: Vault/ with dashboard, templates, and note types (Work Item, Research Note, Decision, Review, Release, Daily Log); Codex-Global/ with shared AGENTS.md, config.toml, and shared agents; Repo-Skills/.agents/skills/ with obsidian-handoff, work-item-intake, app-prd-gate, release-gate, and review-to-actions; Project-App/ AGENTS plus optional .codex/config.toml; Project-Quant/ AGENTS plus .codex/config.toml. APP flow should use research -> PRD -> UX -> architecture -> build -> QA -> release with Obsidian note ids and structured handoffs.
